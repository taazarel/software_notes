# Bi-Implication

$p\leftrightarrow q$ is true if p and q have the same truth value, ergo the opposite of [[XOR]].

| p   | q   | $p\leftrightarrow q$ |
| --- | --- | -------------------- |
| T   | T   | T                    |
| T   | F   | F                    |
| F   | T   | F                    |
| F   | F   | T                    |
