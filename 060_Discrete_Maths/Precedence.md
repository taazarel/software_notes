# Precedence

This is the precedence, order of operations, of [[Negation]]s, [[Conjunction]]s, [[Disjunction]]s, [[Implication]]s and [[Bi-Implication]]s.

- $ Lower number goes first.

| $\neg$            | 1   |
| ----------------- | --- |
| $\land$           | 2   |
| $\lor$            | 3   |
| $\to$             | 4   |
| $\leftrightarrow$ | 5   |
