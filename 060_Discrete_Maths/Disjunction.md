# Disjunction

If either p or q is true, the disjunction will return True.

This is the "or" operator, it is written $\lor$.

This is the truth table for disjunctions given p and q

| p   | q   | $p\lor q$ |
| --- | --- | --------- |
| T   | T   | T         |
| T   | F   | T         |
| F   | T   | T         |
| F   | F   | F         |
